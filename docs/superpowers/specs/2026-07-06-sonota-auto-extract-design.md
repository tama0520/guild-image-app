# 📝記入部分のみモード「その他の優秀台ピックアップ」自動抽出 設計

作成日: 2026-07-06

## 背景・目的

②個別画像の「その他の優秀台ピックアップ」は現在、台番テキスト欄に貼った台だけで画像を作る。フル⑧実行のパイプラインは「その他の優秀台+1,000/+2,000/+3,000枚以上」を自動生成するが、**📝記入部分のみモード**（`_manual_preview_mode`）はパイプラインを回さないため自動抽出がない。

記入部分のみモードで、台番テキストが空欄のとき、ラジオで選んだ閾値（+1,000/+2,000/+3,000枚以上）の「その他の優秀台ピックアップ」を、記入済みの個別画像・並び・末尾の台を除外して自動生成できるようにする。

## スコープ

- **📝記入部分のみモードのみ**（プレビュー `_manual_prev_btn` 生成 ＝ `streamlit_app.py:6984-` と、⑧実行の `_is_manual_mode` ＝ `streamlit_app.py:7756-`）。
- 通常のフルプレビュー/⑧実行（パイプラインが自動生成）と記事用ページは**対象外**。

## UI 仕様

②個別画像の「その他の優秀台ピックアップ」内、台番テキスト欄の**下にラジオボタン**を追加：

- 選択肢：`なし`（デフォルト）／`+1,000枚以上`／`+2,000枚以上`／`+3,000枚以上`
- キー：`sonota_extra_auto_{store}`（`st.radio`）。永続化キー一覧（`_auto_input_keys`）に追加、`on_change=_save_auto_inputs`。

## 挙動（記入部分のみモード）

「その他の優秀台ピックアップ」画像の生成条件を次のようにする：

1. **台番テキストに入力あり** → 従来どおり、その台番だけで画像を作る（ラジオは無視）。
2. **台番テキスト空欄 かつ ラジオ ≠ なし** → 自動抽出で画像を作る：
   - 閾値 `thr` = ラジオ選択（1000 / 2000 / 3000）。
   - **除外機種** `_exc_mac` = ②個別画像の全台（`kojin_zentai_machines`）＋優秀台（`kojin_yushu_machines`）に記入した機種名。
   - **除外台番** `_exc_ban` = 個別機種ピックアップの台番（`_collect_kojin_pick`）∪ 並び台番（`narabi_ranges` ＋ `kojin_narabi_range` / `kojin_narabi2_range` を `ranges_to_bans`）∪ 末尾台番（`suebangai_tail_input_*` / `jug_sue_tail_input_*` から算出。ジャグラー末尾はジャグラー系列に限定）。
   - **対象台** = `df` のうち `機種名 ∉ _exc_mac` かつ `台番 ∉ _exc_ban` かつ `差枚 ≥ thr`。ジャグラーも含める。
   - 台番順にソートして画像1枚を生成。空なら生成しない。
3. **台番テキスト空欄 かつ ラジオ = なし** → 従来どおり何も作らない。

- タイトルは常に「その他の優秀台ピックアップ」（`sonota_extra_title` 欄の値。既定でその文字列）。ピンクバーなし（`_build_machine_img(df, title, None)`）。
- ファイル名 = `{_make_safe_fn(title)}.jpg`。
- スランプ付きページでは、生成画像の台番を `_manual_ban_map` / `_m_exec_ban_map_e` に登録し、既存の記入部分スランプ合成（`_composite_slump_onto_images`）の対象に含める。

## 共通ヘルパー

除外台番の算出（並び・末尾）は記入部分プレビューと⑧実行で共通なので、小ヘルパーで共有する：

```
_manual_sonota_auto_bans(df, store, kojin_zentai_machines, kojin_yushu_machines,
                         narabi_ranges, kojin_narabi_range, kojin_narabi2_range) -> tuple[set[str], set[int]]
```
戻り値 `(_exc_mac, _exc_ban)`。末尾台番は `suebangai_enabled` / `jug_sue_enabled` の session_state を参照して算出。

（YAGNI：ヘルパー化が複雑になる場合は各所インラインでも可。ただしプレビューと⑧実行で除外ロジックを一致させること。）

## エッジ

- 台番テキストとラジオ（≠なし）が**両方**指定された場合は、**台番テキスト優先**（従来動作）。
- 自動抽出結果が0台なら画像を生成しない（警告は既存の記入なし警告に委ねる）。

## 非対象（YAGNI）

- 複数閾値の同時生成（ラジオは1つだけ選択→1枚）。
- 通常フルフロー・記事用ページ。
- ジャグラーの特別扱い（案1：ジャグラーも「その他」に含める）。
